# Sloane Ranger

**Style ID:** `sloane_ranger`  
**Active Window:** 1980–1989  
**Origin:** London / Home Counties, UK  
**Decade(s):** 1980s

## Overview

Upper-class British country-coded staples: waxed jackets, tweed skirts, pie-crust blouses, pearls, and riding-adjacent practical boots.

**Cultural context:** Conservative class signaling; aristocratic heritage branding.

**Aesthetic opposites / contrasts:** Punk; rave.
