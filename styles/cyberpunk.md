# Cyberpunk Fashion

**Style ID:** `cyberpunk`  
**Active Window:** 1995–2000  
**Origin:** Global (film + club influence)  
**Decade(s):** 1990s

## Overview

Black technical layers, long coats, tactical straps, combat boots, and slim wrap sunglasses projecting dystopian digital futurity.

**Cultural context:** Sci-fi media influence; internet-era anxiety; hacker iconography.

**Aesthetic opposites / contrasts:** Preppy / ivy; bohemian revival (90s).
