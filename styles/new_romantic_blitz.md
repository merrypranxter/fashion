# New Romantic / Blitz Scene

**Style ID:** `new_romantic_blitz`  
**Active Window:** 1980–1984  
**Origin:** London, UK  
**Decade(s):** 1980s

## Overview

London's Blitz nightclub (Covent Garden, run by Steve Strange) was the origin point. The scene produced a DIY-costumed glamour: pirate shirts, theatrical makeup, historical pastiche tailoring, androgyny as performance. The Design Museum's Blitz exhibition documents it as a pivotal moment in British style. This is also the incubator for New Romantic pop music's visual language.

**Cultural context:** Post-punk London club culture; fashion as performance art; early MTV image culture.

**Aesthetic opposites / contrasts:** Grunge; punk's confrontational destruction
