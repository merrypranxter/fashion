# UK Casuals / Terrace Fashion

**Style ID:** `uk_casuals_terrace`  
**Active Window:** 1980–1989  
**Origin:** UK football cities  
**Decade(s):** 1980s

## Overview

Expensive European sportswear and clean trainers used as coded status markers on football terraces, deliberately avoiding club-color uniforms.

**Cultural context:** Policing pressure; hooligan identity tactics; import-brand competition.

**Aesthetic opposites / contrasts:** Preppy / ivy; power dressing.
