# Gen X Soft Club

**Style ID:** `gen_x_soft_club`  
**Active Window:** 1997–2000  
**Origin:** Urban nightlife / media  
**Decade(s):** 1990s

## Overview

A muted, comfort-forward branch of late-90s futurism using cool tones, smooth synthetics, and pared-back club silhouettes.

**Cultural context:** Desire for calmer futurism amid Y2K hype; liminal digital aesthetics.

**Aesthetic opposites / contrasts:** Glam metal / hair metal; power dressing.
